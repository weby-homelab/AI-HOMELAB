## 4. Висновки

- **Швидкість:** Модель стабільно видає ~47 ток/с декодування завдяки Native-MTP. Prefill ~330 ток/с (lifetime середнє). Це цілком прийнятно для інтерактивного агента на споживчому GPU 2018 року.
- **Якість коду:** Базова структура даних (LRU з TTL) реалізована коректно. Складніші алгоритмічні задачі (min path у сітці) та імплементація інтерфейсів (TimeMap) — провал. Terminal-Bench — базові shell-команди працюють (`find`, `awk`), складні (openssl cert) — ні.
- **Ресурси:** VRAM заповнена на 88% — модель ледве вміщується в 11 GB. Споживання енергії 112–119 Вт — ефективно для 35B MoE моделі.
- **Методологія:** Результати отримано реальним виконанням коду/команд через OpenAI-сумісний ендпоїнт. Усі метрики швидкості — з Prometheus (`/metrics`) та логів сервера.

## 5. Методологія

- **Генерація:** OpenAI-сумісний ендпоїнт `/v1/chat/completions`, `temperature=0.15`, `reasoning-budget=2048`.
- **LiveCodeBench:** Модель генерує Python-код → виконується у тимчасовій директорії з реальними assert-ами.
- **Terminal-Bench:** Модель генерує bash-команду → виконується у тимчасовій директорії → перевіряється stdout/файли.
- **Швидкість:** Prometheus метрики `llamacpp:prompt_tokens_seconds` та `llamacpp:predicted_tokens_seconds` (lifetime середні).
- **Дата:** 2026-07-14, сервер працює безперервно з 02:40 EEST.
