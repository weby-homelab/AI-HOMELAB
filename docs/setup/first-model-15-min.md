# ⏱️ ГАЙД: Перша локальна LLM за 15 хвилин

Цей гайд допоможе вам розгорнути повноцінну, повністю приватну мовну модель (LLM) з красивим веб-інтерфейсом на вашому власному комп'ютері чи сервері. 

**Безкоштовно. Без хмари. Без цензури. 100% локально.**

---

## 🛠️ Що ми будемо ставити

Ми розгорнемо класичний стек домашньої AI-лабораторії:
1. **Ollama** — рушій інференсу моделей (керує запуском моделей, споживанням пам'яті та GPU).
2. **Open WebUI** — преміальний веб-інтерфейс у стилі ChatGPT/Claude з підтримкою локальної історії, RAG та мультимодальності.

---

## ⚡ Крок 1: Встановлення Ollama

Оберіть вашу операційну систему:

### Linux
Запустіть офіційний скрипт встановлення:
```bash
curl -fsSL https://ollama.com/install.sh | sh
```
*Для запуску на GPU переконайтеся, що у вас встановлено пропрієтарні драйвери Nvidia та `nvidia-container-toolkit`.*

### macOS
1. Завантажте [Ollama-darwin.zip](https://ollama.com/download/Ollama-darwin.zip).
2. Розпакуйте та перетягніть застосунок `Ollama` до папки `Applications`.
3. Запустіть застосунок через Spotlight або Launchpad.

### Windows
1. Завантажте офіційний інсталятор [OllamaSetup.exe](https://ollama.com/download/OllamaSetup.exe).
2. Запустіть інсталятор та слідуйте інструкціям.
3. Ollama автоматично згорнеться в системний трей (Taskbar).

## 🐳 Крок 2: Запуск Open WebUI через Docker

Найпростіший спосіб запустити інтерфейс без конфліктів залежностей — використати Docker. 

Якщо Docker ще не встановлено, поставте його за 2 хвилини:
- **Windows/macOS**: Встановіть [Docker Desktop](https://www.docker.com/products/docker-desktop/).
- **Linux**: `curl -fsSL https://get.docker.com | sh`.

Запустіть контейнер з Open WebUI:
```bash
docker run -d \
  -p 3000:8080 \
  --add-host=host.docker.internal:host-gateway \
  -v open-webui:/app/backend/data \
  -e OLLAMA_BASE_URL=http://host.docker.internal:11434 \
  --name open-webui \
  --restart always \
  ghcr.io/open-webui/open-webui:main
```

## 📥 Крок 3: Завантаження вашої першої моделі

Тепер завантажимо саму модель. Відкрийте термінал (або PowerShell) та виконайте команду залежно від обсягу оперативної пам'яті (RAM/VRAM) вашого комп'ютера:

### 🟢 1. Для слабких ПК (до 8 GB RAM) або енергоощадних мікро-серверів
Використовуйте сучасну та надзвичайно швидку модель **Gemma 4 (E4B)** від Google DeepMind:
```bash
ollama run gemma4:e4b
```

### 🟡 2. Універсальний вибір (16 GB RAM або GPU з 8 GB+ VRAM)
Золотий стандарт для щоденних завдань, програмування та RAG — **Llama 3.1 (8B)** від Meta:
```bash
ollama run llama3.1:8b
```

### 🔵 3. Преміум варіант (32 GB RAM або Mac M-серії з великою об'єднаною пам'яттю)
Потужна модель для складного аналізу та креативного письма — **Mistral Small (22B-Instruct)** або **Gemma 4 (26B MoE)**:
```bash
ollama run gemma4:26b
```

*Після успішного завантаження у терміналі з'явиться інтерактивний чат `>>>`. Напишіть `/exit`, щоб вийти.*

---

## 🎉 Крок 4: Чат через браузер

1. Відкрийте браузер та перейдіть на `http://localhost:3000`.
2. Створіть свій перший локальний акаунт (перший зареєстрований користувач стає адміністратором, всі дані зберігаються **виключно локально** у вашому Docker volume).
3. У верхньому меню оберіть завантажену модель (наприклад, `gemma3:4b`).
4. Насолоджуйтесь швидким та повністю приватним чатом! 🚀

## 💡 Поради для кращого досвіду

* **Промпти українською**: Моделі `gemma3` та `llama3` чудово розуміють українську мову. Якщо модель відповідає англійською, просто додайте в системний промпт: *"Завжди відповідай українською мовою"*.
* **Зупинка контейнера**: Якщо ви хочете вимкнути інтерфейс, виконайте `docker stop open-webui`. Для повторного запуску — `docker start open-webui`.
* **Видалення моделей**: Щоб звільнити місце на диску, видаліть непотрібну модель командою `ollama rm <назва_моделі>`.
