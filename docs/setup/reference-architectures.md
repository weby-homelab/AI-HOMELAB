# 🏗️ AI-HomeLab Reference Architectures: Еталонні архітектури локального ШІ

Цей документ представляє серію еталонних архітектур для побудови домашніх AI-лабораторій різного рівня. Вони орієнтовані на максимальну економічність, автономність та відповідність принципам безпеки проєкту AI-HomeLab.

> [!IMPORTANT]
> Відповідно до меморандуму проєкту, всі архітектурні рішення орієнтовані **виключно** на використання безпечних західних open-source моделей (Meta LLaMA, Google Gemma, Mistral, Microsoft Phi). Використання моделей з РФ та КНР (DeepSeek, Qwen) не тестується та не підтримується.

## ⏱️ Tier 1: Бюджетний локальний ШІ (Mini PC) — $300-500

### 1. Концепція
Цей тієр розрахований на новачків та користувачів із жорстким бюджетом або високим пріоритетом енергоефективності. Замість шумного та важкого ПК використовується ультракомпактний неттоп (Mini PC) на базі сучасних процесорів Ryzen (з потужною вбудованою графікою Radeon 680M/780M) або Intel Gen 12+.

### 2. Специфікація обладнання (BOM)
* **Апаратна платформа:** Mini PC (наприклад, Beelink SER6/SER7, Minisforum UM780 XTX, або аналог Asus ExpertCenter).
* **Процесор:** AMD Ryzen 5 7535HS / Ryzen 7 7840HS (8 ядер/16 потоків).
* **ОЗП:** 32 GB DDR5-4800 (SO-DIMM).
* **Накопичувач:** 1 TB NVMe SSD Gen 4 (швидкість читання >5000 MB/s для швидкого Cold Start).
* **Живлення:** Зовнішній блок живлення (GaN) на 65-100W.

### 3. Програмний стек
* **ОС:** Linux (Ubuntu 24.04 LTS Desktop/Server).
* **Шар інференсу:** **Ollama** (інференс на CPU + часткове прискорення через ROCm для Radeon 780M, або чистий CPU-інференс із фіксованою кількістю потоків).
* **Інтерфейс:** **Open WebUI** (запуск у Docker-контейнері).
* **База даних:** ChromaDB (локальна, in-memory) для RAG.

### 4. Рекомендований вибір моделей
* **Основна LLM:** `gemma3:4b` (Q4_K_M) або `llama3.2:3b` (Q4_K_M).
* **Embedding модель:** `all-minilm` або `nomic-embed-text` (виконується на CPU).

### 5. Метрики та Енергоефективність
* **Швидкість генерації:** 18–25 t/s на моделях до 4B параметрів.
* **Споживання:** 12W у режимі спокою, 35W під навантаженням.
* **Блекаут:** Станція EcoFlow River 2 (256Wh) забезпечить **7-8 годин** безперервної роботи.

## 🤖 Tier 2: Просунутий ШІ & Агентна Інфраструктура (Nvidia GPU) — $1000-1500

### 1. Концепція
Базовий стандарт для розробників штучного інтелекту, що дозволяє запускати повноцінні мультиагентні системи, розробляти RAG-пайплайни та підтримувати локальних асистентів у реальному часі. Основним обчислювальним рушієм є дискретна відеокарта Nvidia з архітектурою Ada Lovelace або Ampere для стабільної роботи CUDA.

### 2. Специфікація обладнання (BOM)
* **Процесор:** Intel Core i5-13400 / Ryzen 5 7600 (6-8 ядер з високою однопоточною швидкістю).
* **Материнська плата:** Чипсет B760 / B650 з підтримкою PCI-e Gen 4.
* **Відеокарта:** **Nvidia RTX 3060 12GB GDDR6** (найбільш економний вхід у 12GB VRAM) або **RTX 4060 Ti 16GB GDDR6** (ідеальний баланс VRAM/TDP).
* **ОЗП:** 64 GB DDR5-5600 (двоканальний режим).
* **Накопичувач:** 2 TB NVMe SSD Gen 4 (читання >7000 MB/s).
* **Блок живлення:** 650W з сертифікатом 80 Plus Gold.

### 3. Програмний стек
* **ОС:** Linux (Ubuntu 24.04 LTS / Debian 12).
* **Оркестрація:** **LiteLLM** (як єдиний OpenAI-сумісний роутер та проксі-сервер) + **n8n** / **LangGraph** (для створення візуальних та кодових агентних робочих процесів).
* **Шар інференсу:** **Ollama** з CUDA-прискоренням.
* **Векторна БД:** **Qdrant** (Docker-контейнер з NVMe-оптимізованим сховищем).
* **Інтерфейс:** **Open WebUI**, підключений до LiteLLM.

### 4. Рекомендований вибір моделей
* **Основна LLM (Reasoning/Coding):** `phi4:14b` (Q4_K_M) або `llama3.1:8b` (Q8_0).
* **Embedding модель:** `nomic-embed-text` (виконується на GPU).

### 5. Метрики та Енергоефективність
* **Швидкість генерації:** 22–35 t/s на моделях 8B-14B параметрів.
* **Споживання:** 65W у режимі спокою, 150-220W під навантаженням (з урахуванням ліміту живлення GPU через Power Limit).
* **Блекаут:** Станція EcoFlow Delta 2 (1024Wh) забезпечить **4-5 годин** роботи в режимі активної генерації.

## ⚡ Tier 3: Максимальна якість & Розподілений інференс (Multi-GPU) — $3000+

### 1. Концепція
Професійне робоче місце для командної розробки, запуску великих моделей (до 70B+ параметрів), хостингу API для невеликих команд та проведення дрібних експериментів із fine-tuning (LoRA). Архітектура будується на використанні кількох графічних процесорів (Multi-GPU) з агресивним розподілом контексту.

### 2. Специфікація обладнання (BOM)
* **Процесор:** AMD Ryzen 9 7900X / Intel Core i9-14900K (12-24 ядер).
* **Материнська плата:** X670E / Z790 з підтримкою кількох слотів PCIe x16 (працюючих щонайменше у режимі x8/x8).
* **Відеокарти:** **2x Nvidia RTX 3090 24GB GDDR6X** (вживані, сумарно 48GB VRAM) або **2x RTX 4090 24GB** (максимальна продуктивність).
* **ОЗП:** 128 GB DDR5 (із підтримкою ECC за можливості).
* **Накопичувач:** RAID 0 з 2x 2 TB NVMe SSD Gen 4 (для максимального throughput при завантаженні моделей).
* **Блок живлення:** 1200W-1600W Titanium/Platinum клас.

### 3. Програмний стек
* **ОС:** Proxmox VE 8+ (як гіпервізор для поділу ресурсів) або чистий Ubuntu Server 24.04 із Docker.
* **Шар інференсу:** **vLLM** (використовує PagedAttention, тензорний паралелізм та динамічне батчування запитів) або **TensorRT-LLM**.
* **Роутинг та Лог-аналітика:** **LiteLLM** (з розподілом навантаження між екземплярами vLLM) + **Langfuse** (для трейсингу та аудиту витрат API).
* **Workflow:** n8n / LangGraph / CrewAI.
* **Векторна БД:** **Qdrant** (кластерний варіант).

### 4. Рекомендований вибір моделей
* **Основна LLM:** `llama3.1:70b` (Q4_K_M або Q5_K_M) або `gemma3:27b` (FP16).
* **Допоміжні LLM (агенти):** `phi4:14b` або `llama3.1:8b`.
* **Embedding модель:** `bge-m3` або `nomic-embed-text`.

### 5. Метрики та Енергоефективність
* **Швидкість генерації:** 15–20 t/s на моделях 70B при паралельних запитах; до 80 t/s на менших моделях.
* **Споживання:** 120W у режимі спокою, 500-850W під навантаженням.
* **Блекаут:** Потребує потужних інверторів (від 3 кВт) та ємних акумуляторних збірок LiFePO4 (від 5 кВт·год). Робота від класичних домашніх EcoFlow можлива лише протягом 1-2 годин.

## 🔌 Рекомендації щодо вибору заліза

1. **VRAM — це король:** Продуктивність ШІ-моделей залежить від швидкості та об'єму відеопам'яті. Завжди купуйте відеокарту з найбільшим об'ємом VRAM, який ви можете собі дозволити. 12GB VRAM — це абсолютний мінімум у 2026 році.
2. **Apple Silicon як альтернатива:** Якщо вашим пріоритетом є безшумність та робота під час тривалих відключень світла — Mac Mini / Studio на чіпах M2/M4 Pro/Max є ідеальними. Завдяки уніфікованій пам'яті (до 192GB) вони запускають величезні моделі, споживаючи всього 25-45W. Але пам'ятайте: швидкість генерації (tokens/s) на Apple Silicon буде нижчою, ніж на дискретних картах Nvidia RTX, через меншу пропускну здатність шини пам'яті порівняно з GPU і відсутність CUDA.
3. **Швидкість SSD:** Не економте на SSD. Квантовані моделі об'ємом 10-40 GB повинні завантажуватися швидко. NVMe SSD Gen 4 дозволяє зменшити час холодного старту моделей до 2-3 секунд.
